# Gustavo Badalotti Portfolio

Portfólio pessoal em Next.js + Tailwind CSS com foco em database engineering, performance de PostgreSQL, animações de scroll e layout bilíngue (EN/PT).

## Stack

- Next.js 16 (App Router)
- React 18
- Tailwind CSS 4
- TypeScript

## Rodando localmente

1. Instale dependências: `npm install`
2. Rode em desenvolvimento: `npm run dev`
3. Abra: `http://localhost:3000`
4. Se estiver fora da pasta do projeto, rode: `npm run dev --prefix /Users/gustavotartas/Projects/gustavo-badalotti-portfolio`

## Build de produção

1. Validar lint: `npm run lint`
2. Gerar build: `npm run build`
3. Testar local em modo produção: `npm run start`

## Checklist

- Atualizar foto em `/public/images/profile.jpg`
- Atualizar PDF do currículo em `/public/resume/cv.pdf`
- Revisar contatos e links em `/src/data/portfolio-data.ts`
- Rodar `npm run lint` e `npm run build`
